# 🤖 Telegram Bot Hosting Platform

بوت تلجرام متقدم لإدارة واستضافة بوتات Python و JavaScript بشكل احترافي.

## ⚡ المميزات الرئيسية

### 📤 رفع وتشغيل الملفات
- دعم ملفات Python (.py) و JavaScript (.js) و ZIP
- تشغيل تلقائي للبوتات بعد الرفع
- إدارة متعددة للبوتات المشتغلة

### 👥 نظام المستخدمين
- نظام اشتراكات مدفوعة
- حدود تحميل مختلفة حسب نوع المستخدم:
  - مستخدم عادي: 3 ملفات
  - مشترك: 15 ملف
  - أدمن: 999 ملف
  - المالك: غير محدود

### 🛡️ لوحة التحكم للأدمن
- إدارة المستخدمين (حظر/فك حظر)
- إدارة الاشتراكات
- إضافة وحذف الأدمن
- بث رسائل جماعية
- إحصائيات تفصيلية
- قفل/فتح البوت

### 🔧 مميزات تقنية
- تثبيت تلقائي للمكتبات المطلوبة
- Keep-Alive باستخدام Flask
- قاعدة بيانات SQLite
- نظام Logs متقدم
- معالجة الأخطاء الشاملة

## 🚀 التشغيل

### المتطلبات
```bash
pip install -r requirements.txt
```

### تشغيل البوت
```bash
python3 telegram_bot.py
```

أو استخدام السكريبت الجاهز:
```bash
chmod +x start.sh
./start.sh
```

## ⚙️ الإعدادات

عدّل المتغيرات التالية في الملف:

```python
TOKEN = 'YOUR_BOT_TOKEN'        # توكن البوت من @BotFather
OWNER_ID = YOUR_OWNER_ID        # معرف المالك الأول
ADMIN_ID = YOUR_ADMIN_ID        # معرف المالك الثاني
YOUR_USERNAME = '@username'      # اسم المستخدم
UPDATE_CHANNEL = 'channel_link'  # رابط قناة التحديثات
```
